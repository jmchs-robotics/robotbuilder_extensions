TimedCommand

What is it?

An action that will be performed for a fixed duration of time. This command acts just like any other command in that you can use it in a command group etc....

Requires
The subsystem that this command requires.
Button on SmartDashboard
Whether or not to automatically place a button on the SmartDashboard.

What you have to do

Add a line of code to the initialize method so that it looks like this. 

    protected void initialize() {
       setTimeout(8);
    }
Make sure to replace the 8 with a value of your choice in seconds.


Next, add another line of code to your isFinished method to it looks like this. 
    protected boolean isFinished() {
       return isTimedOut();
    }
